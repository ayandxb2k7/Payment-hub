##  PaymentHub AI

LIVE DEPLOYED LINK  - https://payment-hub-five.vercel.app/#/login

> Enterprise Payment Infrastructure Platform built for modern businesses.

PaymentHub AI is a production-ready payment management platform that enables businesses to securely manage customers, transactions, invoices, subscriptions, refunds, analytics, and AI-powered financial insights from a single dashboard.

Unlike a traditional e-commerce checkout, PaymentHub AI focuses on the operational side of digital payments, providing enterprise-grade financial management with a clean and modern interface.

# Getting Started

Clone the repository

```bash
git clone https://github.com/ayandxb2k7/Payment-hub.git
```

Go to project

```bash
cd Payment-hub
```

Configure your environment

```env
DB_HOST=
DB_NAME=
DB_USER=
DB_PASS=

PAYPAL_CLIENT_ID=
PAYPAL_SECRET=

GEMINI_API_KEY=
```

Import

```text
database/schema.sql
```

Run

```text
Apache + PHP + MySQL
```

Open

```text
http://localhost/Payment-hub
```
